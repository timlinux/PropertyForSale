# Getting Started

Welcome to PropertyForSale! This guide will help you explore properties and
make the most of our immersive features.

## Browsing Properties

1. Visit the homepage to see featured properties
2. Click "Browse Properties" to see all available listings
3. Use the search and filter options to narrow down your choices

## Viewing a Property

Each property page includes:

- **Hero Images** - High-quality photos of the property
- **Description** - Detailed information about the property
- **Interactive Map** - Explore the property layout
- **3D Tours** - Immersive 360-degree video walkthroughs
- **Floor Plans** - 2D and 3D architectural views

## Creating an Account

Sign in using your preferred social account:

- Google
- Apple
- Microsoft
- GitHub
- Facebook

Once signed in, you can:

- Save favorite properties
- Receive updates on new listings
- Contact property owners directly
